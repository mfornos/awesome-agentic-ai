# Contributing to Awesome Agentic AI

Thank you for your interest in contributing to **Awesome Agentic AI**!
This is a curated list of high-quality resources for building agentic AI systems.

We welcome thoughtful additions, corrections, and improvements.

## What Makes a Good Contribution?

Please ensure that any submission:

- Is clearly relevant to agentic AI systems
  - Goal-directed agents
  - Multi-step reasoning and planning
  - Tool use, memory, coordination, or autonomy
- Represents a maintained, high-quality project or reference
- Is not promotional or purely marketing content
- Adds new value (avoid duplicates or near-duplicates)

## How to Contribute

1. Fork this repository.
2. Create a new branch for your change.
3. Add your entry in the appropriate section:
   - Keep the list alphabetically ordered within each section when possible.
   - Use the existing format for entries.
4. Submit a pull request with:
   - A short description of what you added or changed.
   - A brief justification for why it belongs in the list.
